# Pico Jarvis

Requirement:
* Node.js v18 or later.
* [llama.cpp](https://github.com/ggerganov/llama.cpp) running a model, e.g. [Orca Mini 3B](https://huggingface.co/TheBloke/orca_mini_3B-GGML).

Run:
```
$ node pico-jarvis
```

and then open `localhost:5000`.
